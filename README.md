🌾 FarmCycle
FarmCycle is a full-stack web application designed to connect local farms directly to buyers, making it easier for farmers to sell their products and for buyers to access fresh produce with ease.

🚀 Project Overview
FarmCycle bridges the gap between farmers and consumers by providing a digital marketplace where farm products can be listed, viewed, and purchased.
The platform ensures efficiency, transparency, and accessibility in agricultural trade.

🛠️ Technologies Used
🌐 Frontend
React.js – For building a responsive and dynamic user interface.


🔧 Backend
Node.js & Express – For handling API requests and managing server-side logic.

🗄️ Database
MySQL – For storing user information, product listings, and transactions securely.

📦 Features
Add, edit, and delete farm products
User-friendly interface for buyers and sellers
